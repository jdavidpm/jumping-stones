Problem

A frog wants to cross the rivers but doesn't want to get wet. There is a path made out of N stones that leads to the other side. In order to move from one stone to another, the frog has to jump, and in each jump the frog can move from 1 to k stones.
In how many ways can the frog cross the river?

Example:

If the path had 4 stones, and k equals to 3, the frog could cross the river in 7 different jumps sequences:
     ·  [1, 1, 1, 1]
     ·  [1, 1, 2]
     ·  [1, 2, 1]
     ·  [2, 1, 1]
     ·  [2, 2]
     ·  [1, 3]
     ·  [3, 1]

How is this problem solved?

Dynamic Programming

PD is a technique that helps us reduce running time of an algorithm assuming that the problem has superimposed problems and optimal substructures.

This technique has two different approaches:

     ·  Top-down
     ·  Bottom-Up
